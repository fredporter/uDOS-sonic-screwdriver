# Rescue Module

This module represents Sonic's rescue and recovery lane.

Current implementation lives across:

- `scripts/collect-logs.sh`
- `scripts/verify-usb-layout.sh`
- `scripts/tinycore-bootlog.sh`
- `memory/sonic/` runtime artifacts and logs
- `courses/03-portable-dev-systems/`

Educational role:

- teach recovery-first workflows
- teach how portable Sonic media can diagnose and recover a target system
- teach safe evidence collection before destructive repair steps
