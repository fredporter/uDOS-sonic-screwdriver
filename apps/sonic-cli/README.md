# Sonic CLI

This is the public CLI surface for Sonic.

Current implementation:

- `apps/sonic-cli/cli.py`
- installable console entrypoint: `sonic`

Core commands:

- `help`
- `commands`
- `start`
- `status`
- `test`
- `doctor`
- `plan`
- `run`
- `serve-api`
- `serve-mcp`

Educational role:

- teach reviewed planning before destructive apply
- teach explicit command surfaces for provisioning workflows
- teach starter CLI handoff before opening the GUI
